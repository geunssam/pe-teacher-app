아래를 코드로 구현해라. 추측하지 말고 명시된 요구만 반영한다.

요구사항:
1) structure를 activity 개념으로 취급하되 런타임 호환 유지
2) 제목-본문 불일치 차단 계약 강제
3) 6학년 축구 전용: 1(조건) -> 2(수업확정), 2단계 인라인 편집+바로 저장
4) 2단계/확정 화면에 적용 변형 해설 표시

검증:
- npm run build 통과
- 제목에 표시된 modifier 규칙이 본문에 반드시 존재
- 2단계 편집 반영 저장 확인

출력:
- 변경 파일 목록
- 핵심 diff
- 테스트 결과
- 남은 리스크
