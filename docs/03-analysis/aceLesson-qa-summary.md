## 2. 우선순위 수정 큐

### Tier 1: 즉시 재작성 — 3학년 스포츠 (18개)

**심각도: 최고** | 21개 중 18개가 보일러플레이트

3종 템플릿이 기계적으로 복사된 구조:
- **Template A** (9개): "동작 분해 연습", "상황 변형 연습", "정확도 확인" — 이론~실기 관계없이 동일
- **Template C** (8개): "핵심 동작 드릴", "동작-규칙 결합", "축소판 전략 미션" — 피구/축구/농구 모두 동일
- **Template E** (2개): 평가 활동에 구체적 체크리스트 누락 (추가 필요)

**가장 심각한 사례**: `g3s_sports_meaning` — 이론 수업(영상 시청 + 스포츠 분류)인데 "무릎 몸통 정렬", "안전거리 확보" 등 운동장 실기 용어가 그대로 삽입됨

**참조 모델**: `g3s_throw_catch_marker_quest` (유일한 PASS) — 교사 커스텀 변형안으로 드릴이 활동 동작을 직접 지칭하고 스캐폴딩에 구체적 수치 포함

### Tier 2: 즉시 재작성 — 3학년 운동 보일러플레이트 (8개)

**심각도: 높음** | 모두 Template C 구조

대상: g3m_sup_exercise_tag, g3m_sup_cup_flip_tic_tac_toe, g3m_sup_gagaball, g3m_sup_classroom_dodgeball, g3m_sup_pearl_shell, g3m_sup_walking_bingo, g3m_sup_secret_tag, g3m_sup_dice_tag

공통 문제:
- acquire 드릴이 제네릭 ("핵심 동작 드릴") — 교사가 어떤 동작을 연습시켜야 하는지 알 수 없음
- engage 게임 규칙이 "점수 기반 토너먼트" 한 줄 — 각 게임 고유 규칙 완전 누락
- metaQuestion, feedback, safetyNote가 8개 모두 동일

### Tier 4: 부분 수정 (26개)

학년별 주요 이슈 유형:

**3학년 운동 (1개)**:
- `g3m_sup_fruit_monster`: 과일 호출 반응 연습의 이동 목표 지점 불분명

**4학년 스포츠 (7개)** — 공통 패턴:
- 성취기준 가치 덕목([4체02-09] 규칙 준수) 피드백 미반영 (3건)
- 스캐폴딩 목표치 과도 (원터치 패스 연속 5회 등) (2건)
- 장비 목록↔aceLesson 불일치 (하키스틱 미사용, 풍선 누락) (2건)

**5학년 스포츠 (8개)** — 공통 패턴:
- 시간 배분 빠듯 (이어달리기 engage 7분에 30명 운영 등) (3건)
- acePhase 분류 재검토 (태권도 겨루기 E→C, 매트 작품발표 성취기준) (2건)
- 드릴 세부 설명 보강 (배구형 게임 acquire 7분 등) (3건)

**6학년 스포츠 (8개)** — 공통 패턴:
- 성취기준 누락 ([6체02-10] 의지와 인정, [6체02-11] 협력과 배려) (3건)
- 스케이팅 활동 영역 분류 일관성 (기술형 vs 생태형) (2건)
- 안전 이슈 보강 (짐볼 카약, 플루크 보겐) (2건)
- 스마트폰 촬영 현실성 (학교 정책 제한) (1건)

### 보일러플레이트 전형 (Worst Case)

```
// Template C — 모든 활동에 동일하게 들어가는 텍스트
acquire.drills[0]: "핵심 동작 드릴 — {활동명}의 핵심 동작을 3단계로 나눠 설명하고 따라 하게 한다"
acquire.drills[1]: "동작-규칙 결합 — 핵심 동작을 적용하는 상황을 만들어 연습"
challenge.missions[0]: "축소판 전략 미션 — 작은 코트에서 3:3 경기"
engage: "점수 기반 토너먼트로 정식 운영"
```

**문제점**: "핵심 동작"이 무엇인지, "3단계"가 구체적으로 뭔지, "상황"이 어떤 상황인지 전혀 명시되지 않아 교사가 이 수업안을 보고 수업을 진행할 수 없음.

## 4. 권장 수정 순서

```
[1단계] 3학년 스포츠 Template A 9개 재작성          ← 가장 많고 가장 심각
[2단계] 3학년 스포츠 Template C 8개 재작성          ← Tier 1 완료
[3단계] 3학년 운동 Template C 8개 재작성            ← Tier 2 완료
[4단계] 3학년 스포츠 Template E 2개 + 기타 REWRITE 3개  ← Tier 3 완료
[5단계] MINOR_FIX 26개 부분 수정                   ← Tier 4
```

### 재작성 시 참조할 골든 패턴

1. **드릴명 = 실제 동작**: "핵심 동작 드릴" X → "인사이드 킥 4단계 연습" O
2. **구체적 수치/거리/시간**: "반복 연습" X → "5m 앞 콘에 5회 연속 패스" O
3. **활동 고유 규칙 명시**: "토너먼트 운영" X → "3:3, 코트 8×5m, 공 2개, 3분 라운드" O
4. **스캐폴딩 실질적 차이**: "더 쉽게/어렵게" X → "거리 3m→5m, 공 소프트볼→축구공" O
5. **metaQuestion 활동 연결**: "어떤 전략이 효과적이었나?" X → "상대가 왼쪽으로 올 때 오른발 인사이드 킥이 효과적이었던 이유는?" O
