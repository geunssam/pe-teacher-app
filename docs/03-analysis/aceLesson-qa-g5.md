### [g5s_freestyle_swim] "자유형으로 빠르게 헤엄치기" (acePhase: A)

- **판정**: REWRITE_NEEDED
- **성취기준**: [6체02-01], [6체02-02]
  - [6체02-01]은 기술형 스포츠의 **의미와 유형 파악**인데, 이 수업에서 기술형 스포츠의 의미/유형을 학습하는 시간이 intro에 1줄뿐이고 대부분 동작 연습임. [6체02-02] (기본 기능 수행)에만 해당.
  - [6체02-03] (기본 전략 수행)이 빠져 있는데, challenge의 미션 3(릴레이 시뮬레이션)은 전략적 요소가 없으므로 이는 적절함.
- **실용성 문제**:
  - **가장 큰 문제**: 수영장이 없는 학교(대다수 초등학교)에서 매트 위 크롤 동작 시뮬레이션을 40분간 진행하는 것은 현실적으로 매우 어려움. acquire 10분 + challenge 12분 + engage 7분 = 29분간 매트 위에 엎드려 팔다리를 움직이는 활동은 학생 흥미를 유지하기 극히 어려움.
  - engage의 "수영 동작 빙고" 게임은 창의적이나, 매트 위 동작만으로는 실제 수영 기능 향상과의 연결성이 매우 약함.
  - **드릴 구체성 부족**: "팔+발 협응 연습" 드릴에서 "매트에 엎드려 팔 돌리기와 발차기를 동시에 수행"이라고만 되어 있는데, 매트 위에서 크롤 팔 돌리기를 물리적으로 수행하기 어려움 (팔이 매트에 걸림). 실제 수행 방법에 대한 구체적 지시가 필요.
  - challenge 미션 1 "30초간 유지" -- 매트 위 크롤 동작을 30초간 유지하는 것은 체력적으로 5학년에게 적합하나, 동기 부여 측면에서 지루함.
- **피드백 문제**:
  - "팔을 돌릴 때 팔꿈치가 먼저 나왔어?" -- 매트 위에서는 이 피드백이 물에서와 다르게 작용함. 매트 위에서의 자세와 물에서의 자세 차이를 반영한 피드백이 필요.
  - scaffoldDown "의자에 앉아 발목만 까딱까딱 움직인다" -- 지나치게 단순화되어 발차기 기능 습득에 도움이 되지 않음.
- **개선 방향**:
  - 수영장이 없는 학교 버전과 있는 학교 버전을 분리하거나, 매트 시뮬레이션은 acquire(10분)만으로 제한하고 challenge/engage에서는 수영 관련 전술 게임(물 대신 볼 릴레이 등)으로 전환.
  - 또는 수업 전체를 "수영의 이해 + 육상 기본 동작"으로 재설정하여, 이론(10분) + 육상 동작(15분) + 수영 관련 퀴즈/보드게임(10분) 구조로 변경.

  - engage의 "영법 퀴즈 릴레이" -- "해당 동작을 수행하며 반환점을 돌아오는 릴레이"는 매트 위 수영 동작을 하며 걸어가는 것인데, 수행 방법이 불명확 (서서 팔 동작을 하며 걷기? 엎드려서?).

  - "글라이딩 때 진짜 2초를 셌어?" -- 매트 위에서 글라이딩 체험은 실제 물에서의 글라이딩 감각과 완전히 다름. 이 피드백이 실제 수영 기능 향상에 기여하는지 의문.
  - scaffoldUp/scaffoldDown은 매트 시뮬레이션 범위 내에서 잘 구분되어 있으나, 본질적 한계가 있음.
- **개선 방향**: 자유형 활동과 동일. 매트 시뮬레이션 비중을 줄이고, 평영의 원리 이해 + 관련 체력 훈련(유연성, 하체 근력) + 물안전 교육을 결합한 수업으로 재구성.

## PASS (통과)

### [g5s_long_jump] "다양한 방법의 멀리뛰기" (acePhase: A)
- 통과 근거: 기록형 기술 스포츠의 A단계 수업으로 완벽하게 구성. 제자리뛰기 → 3보 도움닫기 → 착지 연습의 단계적 드릴이 우수하고, 짝 피드백 시스템(착지 자세 확인)이 현실적. 시간 배분(5+10+12+7+6=40분) 정확. scaffolding이 멀리뛰기 동작에 정확히 맞춤 설계됨.

### [g5s_taekwondo_basic] "태권도 기본 동작" (acePhase: A)
- 통과 근거: 투기형 스포츠의 기본 기능 습득 수업으로 적절. 서기 → 막기 → 지르기 → 앞차기 → 태극 1장의 흐름이 교육과정 순서에 맞음. 안전 규칙("상대를 직접 가격하지 않기")이 명확하고, 기합의 교육적 의미를 질문으로 다룸. 태극 1장 8개 동작을 4개씩 나누어 가르치는 구조가 5학년 수준에 적합. 시간 배분 정확.

### [g5s_rugby_basic] "럭비형 게임 기본 기능" (acePhase: A)
- 통과 근거: 영역형 전략 스포츠의 기본 기능 습득으로 적절. "앞으로 패스 금지"라는 럭비 고유 규칙을 metaQuestion에서 효과적으로 다룸. 공 안고 달리기 → 옆 패스 → 패스 릴레이 → 태그 피하기의 점진적 구성이 우수. 태그 럭비로 안전성 확보. 시간 배분 정확.

### [g5s_rugby_game] "럭비형 게임 하기" (acePhase: C)
- 통과 근거: 전 차시(기본 기능)에서 배운 기술을 경쟁 상황에 적용하는 C단계로 적절. acquire 7분으로 빠른 복습 후 바로 게임 진입하는 구조가 효율적. "동그라미 차지하기" 변형 게임이 창의적이고, 작전 타임 + 재도전 구조가 전략적 사고를 유도. 꼬리잡기 럭비 리그전(전반+작전+후반)이 실전적. 시간 배분(5+7+10+12+6=40분) 정확.

### [g5s_soccer_basic] "축구형 게임 기본 기능" (acePhase: A)
- 통과 근거: 발 안쪽 패스와 공몰기라는 축구 핵심 기본기에 집중한 A단계 수업. "꽃이 피었습니다 공몰기 놀이"가 매우 창의적이고 실제 수업에서 검증된 활동. "바람개비 패스"(사각형 패스 돌리기)도 구체적이고 실용적. scaffolding이 정확함(5m → 지그재그 / 3m → 5m). 시간 배분 정확.

### [g5s_soccer_game] "축구형 게임 하기" (acePhase: C)
- 통과 근거: "골대 2개" 변형 규칙이 공간 활용 전략을 자연스럽게 유도하는 탁월한 설계. 1차전 → 작전 타임 → 2차전의 전략 학습 구조가 체계적. 원터치 패스, 2 vs 1 슈팅 등 acquire 드릴이 실전적. 리그전 운영 방식이 상세. 시간 배분(5+7+10+12+6=40분) 정확.

### [g5s_field_game] "발야구형 게임" (acePhase: C)
- 통과 근거: 한국 초등 전통 인기 종목인 발야구를 필드형 전략 스포츠로 적절히 구성. 차기/받기/던지기 기본기 → 3이닝 연습 게임 → 작전 토의 → 5이닝 정식 게임의 흐름이 매우 현실적. 공격 전략(세게 차기 vs 빈 곳 노리기)과 수비 전략(포지션 배치)을 구분한 것이 교육적. 8-15인 팀 구성이 실제 학급 규모에 적합. 시간 배분 정확.

### [g5s_climbing] "클라이밍 활동" (acePhase: A)
- 통과 근거: 시설 유무에 따라 "클라이밍 월 또는 늑목" 대안을 제시하여 현실적. 그립 3종류(크림프/오픈핸드/핀치)와 트래버스 개념이 5학년 수준에서 충분히 소화 가능. "클라이밍 빙고 챌린지"가 게임 형태의 기능 연습으로 창의적. "팔 힘보다 발 위치가 중요"라는 핵심 원리를 metaQuestion으로 효과적으로 다룸. 안전 규칙(높이 2m 제한, 매트 필수)이 명확. 시간 배분 정확.

### [g5s_orienteering] "오리엔티어링" (acePhase: C)
- 통과 근거: 지도 읽기 + 나침반 + 코스 탐색이라는 생태형 스포츠 고유의 기능/전략이 체계적으로 구성됨. 교실 미니 오리엔티어링 → 학교 주변 실행의 점진적 확장이 안전하면서도 실전적. "결과 발표 대회"에서 루트 전략을 비교하는 것이 전략적 사고를 잘 유도. "모둠 단위 이동" 안전 규칙이 명확. 시간 배분(5+12+10+7+6=40분) 정확.
